XP
Gain xp from MANY THINGS
Player has XP array storing all instances of XP gain, which are objects:
{
    Amount: The amount of XP gained. When the amount of all XP instances reaches a threshold, you level up,
    Source: How you got the XP. This will inform your perk options on level up,
    Weight: how heavily to weighh this instance of XP gain when determining level up perks
}

or, better... create class Leveler.
attributes:
XP - float, number of xp gained
Threshold = int, xp required for levelup
